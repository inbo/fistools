# Boswachterijen

Spatiale en andere informatie (o.a. telefoonummers) van de
boswachterijen van ANB.

## Usage

``` r
boswachterijen
```

## Format

### `boswachterijen`

En sf data frame with 98 rijen and 11 kolommen per jaar:

- Regio:

  Beheerregio

- Naam:

  Naam van de boswachter

- telefoon:

  Telefoon nr van de boswachter

## Source

<https://www.who.int/teams/global-tuberculosis-programme/data>
