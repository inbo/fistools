# CLC18_BE

Corine landcover of Belgium (2018)

## Usage

``` r
CLC18_BE
```

## Format

datapackage

- ID:

  Unieke identificatie polygonen

- CODE_18:

  CORINE landcover code 2018

- LABEL_18:

  CORINE landcover label 2018

- LABEL_Grouped:

  Lijst van media-urls van de geselecteerde deployments

- geometry:

  Geometrie kolom met polygonen

- ...:

  Overige kolommen
