# drg_example

Subset of Drongengoed Agouti export for testing of functions

## Usage

``` r
drg_example
```

## Format

datapackage

- deployments:

  Lijst van geselecteerde deployments

- observations:

  Observaties van de geselecteerde deployments

- media:

  Lijst van media-urls van de geselecteerde deployments

## Source

<https://www.agouti.eu>
